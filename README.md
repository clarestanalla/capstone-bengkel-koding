# Customer Churn Prediction

## Deskripsi

Aplikasi ini digunakan untuk memprediksi apakah pelanggan akan melakukan churn menggunakan model Machine Learning Logistic Regression.

## Model

- Logistic Regression
- StandardScaler
- Accuracy : 0.84
- Recall : 0.28
- F1 Score : 0.32

## Cara Menjalankan

Install library

```bash
pip install -r requirements.txt
```

Jalankan

```bash
streamlit run app.py
```

Upload file CSV untuk melakukan prediksi.
